# final-boss-network-monitor
Real-time network traffic monitoring and analysis tool built with Python and Scapy.


# Final Boss - Ağ Trafiği İzleme Aracı

Final Boss, Python ve Scapy kullanılarak geliştirilmiş gerçek zamanlı bir ağ trafiği izleme ve analiz uygulamasıdır.  
Uygulama, ağ üzerinden geçen paketleri yakalayarak kullanıcıya anlaşılır ve kullanıcı dostu bir arayüz üzerinden sunmaktadır.

---

Projenin Amacı

Bu projenin amacı, kullanıcıların ağ trafiğini daha anlaşılır ve kolay bir şekilde inceleyebilmesini sağlamaktır.
Proje, eğitim ve temel ağ analizi süreçlerinde kullanılabilecek sade ve kullanıcı dostu bir çözüm sunmayı hedeflemektedir.

---

## Özellikler

- Gerçek zamanlı paket izleme
- TCP / UDP / ICMP / ARP / DNS / HTTPS tespiti
- Protokol bazlı filtreleme
- Port analizi
- Canlı arama sistemi
- Sütun bazlı filtreleme
- Paket detaylarını görüntüleme
- CSV formatında dışa aktarma
- Trafik satırlarını kopyalama
- Modern karanlık tema arayüzü

---

## Kullanılan Teknolojiler

- Python
- Scapy
- CustomTkinter
- Tkinter
- Threading

---

## Uygulama Arayüzü

Uygulama kullanıcıya gerçek zamanlı bir trafik izleme paneli sunmaktadır.  
Bu panel üzerinden kullanıcılar:

- Gelen ve giden paketleri görüntüleyebilir
- Protokol ve port analizleri yapabilir
- Trafiği filtreleyebilir
- Paket verilerini dışa aktarabilir
